### 웹 로봇이란?
- 웹 로봇은 사람과의 상호작용 없이 연속된 웹 트랜잭션을 자동으로 수행하는 소프트웨어를 지칭한다.
  - 많은 로봇이 웹 사이트들을 떠돌아다니며 컨텐츠를 조회하거나 링크를 따라가는 방식으로 발견한 데이터를 처리한다.
  - 이러한 로봇들은 모두 자동으로 웹 사이트들을 탐색하며, 방식에 따라 크롤러나 스파이더 또는 웜 및 봇 등의 용어로 지칭된다.
- 웹 로봇은 예를 들어 다음과 같은 작업들을 수행할 수 있다.
  1. 월드 와이드 웹의 규모와 진화와 관련한 통계 정보를 수집
  2. 검색 데이터베이스를 구성하기 위해 발견한 모든 문서를 수집하는 검색엔진 봇

### 크롤러와 스파이더
- 웹 크롤러는 임의의 페이지를 하나 조회한 후, 그 페이지가 참조하는 모든 페이지를 다시 조회하는 식으로 재귀적으로 동작한다.
  - 이렇듯 링크에 따라 웹을 순회하며 재귀적으로 동작하는 모습이 마치 기어가는 것 같아 크롤러 또는 스파이더라는 용어로 지칭한다.
  - 반면, 크롤러가 수집해야할 대상은 매우 많기에 이들은 특성상 복잡한 로직을 가질 수 밖에 없다.
- 검색엔진은  웹을 순회하며 마주치는 모든 문서를 수집하기 위해 크롤러를 사용하며, 이렇게 수집된 문서는 후처리되어 검색 가능한 데이터베이스에 등록된다.
